# 用户信息字段说明

本 Demo 的字段、选项和公开层级严格依据 `投资人及创业者信息.docx`。界面固定分为三个模块：

| 模块 | 规则 |
| --- | --- |
| 公开信息 | 已填写内容一定进入公开 Agent Card |
| 选择性公开信息 | 默认公开；用户可为每个字段单独关闭公开 |
| 不公开信息 | 一定不进入对方 Agent Card 或公共结果，只供本方 Agent 内部使用 |

## 投资人字段

### 公开信息

| 字段 | 输入方式 / 可选项 |
| --- | --- |
| 个人姓名 | 文本 |
| 机构名称 | 文本 |
| 机构类型 | VC基金；CVC/产业资本；政府引导/产业基金；个人投资人；市场化天使基金/VC基金/PE基金；并购基金 |
| 职务 | 文本 |
| 关注赛道 | 芯片&半导体；人工智能&算力；电子硬件；机器人；通信技术；新能源技术；汽车&新能源汽车；军工与国防；机械设备；创新药&CXO；医疗器械；新材料；消费；互联网；其他 |
| 投资阶段 | 文本 |
| 其他说明（根据自己想法，自由补充） | 多行文本 |

### 选择性公开信息

| 字段 | 输入方式 / 可选项 |
| --- | --- |
| 机构规模 | 文本 |
| 个人常驻地 | 文本 |
| 今年计划投资规模 | 文本 |
| 单笔投资规模 | 文本 |
| 目前投资状态 | 积极寻找投资机会；审慎出手；暂时不投 |
| 加入虚拟社区的近期活动目标 | 找投资项目；为已投项目找新资金；为已投项目拉业务；为已投项目找团队；顺手做做FA |
| 具体投资方向 | 多行文本 |

### 不公开信息

| 字段 | 输入方式 / 可选项 |
| --- | --- |
| 最看重的信号 | 市场规模&行业趋势；竞争格局；团队履历&完整性；股权结构（控制权、核心技术人员股份）；技术壁垒&领先性；产品和服务；商业模式合理性；目前商业化阶段&业务数据（研发阶段？产品送样？批量销售？）；融资方案及资金使用计划；其他（合规风险、可比公司情况、荣誉资质） |
| 投资禁忌 | 某些行业；政策风险项目；项目所在地；项目发展阶段；创始人风险（不全职、低学历或年龄过大）；公司负面舆情；其他 |
| Agent报告形式 · 报告周期 | 每天；每两天 |
| Agent报告形式 · 报告方式 | 报告所有交流活动详情；报告所有交流活动简介；只报告认为有价值的交流 |

## 创业者字段

### 公开信息

| 字段 | 输入方式 / 可选项 |
| --- | --- |
| 姓名 | 文本 |
| 公司全称 | 文本 |
| 一句话项目介绍 | 多行文本 |
| 项目亮点 | 多行文本 |
| 核心产品或服务 | 多行文本 |
| 所属赛道 | 文本 |
| 融资轮次 | 文本 |
| 其他说明（根据自己想法，自由补充） | 多行文本 |

### 选择性公开信息

| 字段 | 输入方式 / 可选项 |
| --- | --- |
| 加入虚拟社区的近期活动目标 | 找投资；找人才；寻找业务机会；其他 |

| 业务需求（自由填写） | 多行文本 |
| Agent报告形式 · 报告周期 | 每天；每两天；每周 |
| Agent报告形式 · 报告方式 | 报告所有交流活动详情；报告所有交流活动简介；只报告认为有价值的交流 |
