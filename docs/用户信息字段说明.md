# 用户信息字段说明

本文档依据当前 Demo 的资料表单划分整理。界面将字段分为三部分：

| 分区 | 说明 | 对方可见 |
| --- | --- | --- |
| 公开身份资料 · Agent Card 白名单 | 保存后进入公开 Agent Card，双 Agent 模拟开始时发给对方（未核验自报声明） | ✅ 是 |
| 内部初筛资料 · 必填但不公开 | 仅进入本方 Agent 用户层，不进入对方 Agent Card | ❌ 否 |
| 高级内部画像 · 选填 | 同上；敏感细节建议通过私有文件补充 | ❌ 否 |

字段定义来源：`lib/defaults.ts`（`FIELD_DEFINITIONS`、`AGENT_CARD_FIELD_KEYS`）。
